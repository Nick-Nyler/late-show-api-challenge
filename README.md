# Late Show API

A Flask REST API for managing a Late Night TV show system, built with MVC architecture, PostgreSQL, and JWT authentication.

## Setup Instructions

### Prerequisites
- Python 3.8+
- PostgreSQL
- Pipenv
- Postman
- Git
